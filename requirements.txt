#
# Prerequisites:
# - Python 3.9+ (developed with Python 3.11)
# - Windows OS (for PyWin32 overlay functionality)
# - 2GB+ RAM (for PaddleOCR models)
# - NVIDIA GPU with CUDA (optional, for GPU acceleration)
#
# Installation:
# 1. Create virtual environment: python -m venv venv
# 2. Activate environment: venv\Scripts\activate (Windows)
# 3. Upgrade pip: python -m pip install --upgrade pip
# 4. Install requirements: pip install -r requirements.txt
#
# Additional Setup:
# - Install Tesseract OCR (optional backup engine)
# - Configure OpenAI API key in .env file
# - Ensure RTSViewer.exe is available in system_files/
