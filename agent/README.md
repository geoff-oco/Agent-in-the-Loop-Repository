# Agent Components

This folder contains the core components for the Agent in the Loop system.

## Subfolders

- **action_execution/** - Components for executing actions in the RTS environment
- **decision_logic/** - Core decision-making algorithms and logic
- **integration/** - Integration components for connecting different system parts
- **screen_reading/** - Components for processing and interpreting visual game state
- **tests/** - Test files for agent components
- **visualisation/** - Components for visualizing agent behavior and performance
